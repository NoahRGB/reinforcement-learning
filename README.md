# Reinforcement Learning

Implementations of various reinforcement learning algorithms
- K armed bandits (following [Sutton & Barto Reinforcement Learning: An Introduction](http://incompleteideas.net/book/the-book-2nd.html))
- Maze learning (basic pygame maze environment for practicing algorithms)
- Gymnasium (solutions to different [Farama Gymnasium](https://gymnasium.farama.org/) environments)
